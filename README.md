# Half-Life theme

A simple theme for Obsidian app. Based on a theme I use for programming in Rider.

Inspired by daylerees @ https://github.com/daylerees/colour-schemes

![Screenshot](/images/screenshot.png)


## How to install

1. In Obsidian, open **Settings** > **Appearance** > **Themes** > **Manage**
2. Search for `Half-Life`
3. Click on `Install and use`
